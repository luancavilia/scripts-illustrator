# Scripts para Adobe Illustrator

Coleção de scripts desenvolvidos para automatizar tarefas repetitivas no Illustrator.

## Como usar
1. Baixe o arquivo `.jsx` desejado.
2. No Illustrator, vá em `File > Scripts > Other Script...`
3. Selecione o arquivo baixado.

## Scripts disponíveis
* **AjusteCMYK_Completo.jsx:** Altera a porcentagem de preto (K) em objetos CMYK mantendo as outras cores.
* **EtiquetarCores.jsx:** Cria uma legenda com a cor CMYK abaixo de cada objeto selecionado.
